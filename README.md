# ğŸ”¹ Breadth First Search (BFS)

Traverses the graph level by level.

Uses Queue (FIFO).

Visits all adjacent nodes first.

Useful for shortest path in unweighted graph.

Time Complexity: O(V + E).

Space Complexity: O(V).

# ğŸ”¹ Depth First Search (DFS)

Traverses the graph depth wise.

Uses Stack or Recursion.

Goes deep into one branch before backtracking.

Useful for cycle detection and path finding.

Time Complexity: O(V + E).

Space Complexity: O(V).
